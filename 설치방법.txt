  바둑 (9·13·19) — 핸드폰 앱 설치 방법
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

이 폴더는 설치형 웹앱(PWA)입니다.
아래 파일이 모두 함께 있어야 정상 동작합니다.

  index.html        ← 게임 본체
  manifest.json     ← 앱 정보(이름/아이콘)
  sw.js             ← 오프라인 동작
  icons/            ← 앱 아이콘들


■ 가장 쉬운 방법 — 무료 호스팅에 올리기 (권장)

  1) 이 폴더 전체를 아래 중 한 곳에 올립니다.
     · Netlify Drop : https://app.netlify.com/drop
       (폴더를 브라우저 창에 드래그만 하면 주소가 생깁니다)
     · GitHub Pages / Vercel / Cloudflare Pages 도 가능

  2) 생성된 https:// 주소를 핸드폰 브라우저에서 엽니다.

  3) 홈 화면에 추가
     · 아이폰(Safari) : 하단 [공유] → [홈 화면에 추가]
     · 안드로이드(Chrome) : 우측 상단 ⋮ → [앱 설치] 또는
                            [홈 화면에 추가]

  ※ 설치 안내 문구가 시작 화면 하단에 자동으로 나타납니다.


■ 참고
  · PWA 설치·오프라인 기능은 https:// 주소에서만 완전히 동작합니다.
  · index.html 파일 하나만 열어도 게임은 정상 플레이됩니다
    (단, 이때는 "홈 화면 앱 설치"는 제한될 수 있습니다).


■ 게임 사용법
  · 새 대국 : 바둑판 크기(9×9 / 13×13) · 상대(컴퓨터/2인)
             · 난이도(초/중/고급) · 내 돌 색 선택
  · 교차점을 눌러 착수, 소리 아이콘으로 효과음 on/off
  · 힌트 : 곤경에 처했을 때 다음 한 수를 추천(금색 링으로 표시)
  · 형세 : 대국 도중 아무 때나 눌러 예상 집수·우열을 계산
           (반상에 세력 음영 + '흑/백 O집 우세' 요약, 근사치)
  · 단수(아타리)가 되면 경고음과 "단수!" 표시로 위기를 알림
  · 무르기 / 한 수 쉼(pass) / 기권 버튼 제공
  · 양쪽이 연속으로 한 수 쉬면 집 계산(덤 6.5)으로 승부 결정
